# Glint-Chatbot 🤖

Glint-Chatbot is an AI-powered chatbot built using Gemini 2.0 technology, designed to provide interactive and intelligent conversations with users. This project showcases a smart, responsive web-based chatbot interface that leverages advanced AI for seamless user engagement.

## Features

- Powered by Gemini 2.0 AI model for advanced natural language understanding
- Responsive and user-friendly chat interface
- Real-time conversations with AI responses
- Easy to customize and extend

## Technologies Used

- Gemini 2.0 AI
- JavaScript
- HTML5 & CSS3

1. Clone the repository:
git clone https://github.com/lordhanya/glint-frontend.git

## Usage

- Type your queries or messages in the chat input.
- The chatbot responds in real-time, powered by Gemini 2.0.
